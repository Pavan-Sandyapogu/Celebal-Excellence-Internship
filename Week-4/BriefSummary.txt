As part of this project, 
we created an Azure Resource Group and a Storage Account. We then created a Blob Container and uploaded a CSV file. Next, we set up Azure Data Factory, configured a Linked Service to connect to Blob Storage, and created the source and destination datasets. We used the Get Metadata activity to validate the file metadata and the Copy Data activity to copy data from the source CSV file to the destination file. After successfully executing the pipeline, we verified that the output file was created in Blob Storage. Finally, we assigned the required IAM roles and granted Azure Data Factory access to the Storage Account using the 'Storage Blob Data Contributor' role.
